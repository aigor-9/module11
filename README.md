# Проект: Динамический список фруктов.

Разработано в качестве практического задания на курсе «Fullstack веб-разработчик на JavaScript и PHP». Модуль — Базовый JS. Циклы и массивы.

## Используемые технологии

- HTML
- CSS
- JavaScript

## Цель проекта

Создание динамического списка фруктов

## Результат

- динамически создан список фруктов
- возможность перемешивать список
- фильтрация элементов списка
- возможность сортировать сипсок
- возможность добавлять новые фрукты
